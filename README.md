# УрФУ — абитуриенты по направлениям

Одностраничное приложение для анализа заявлений приёмной кампании УрФУ: конкурс по направлениям, рейтинг по баллам, первый приоритет и позиция «где я» по своему баллу.

**Сайт:** https://omaul.github.io/urfu-entrant/

## Как пользоваться

1. Откройте сайт и загрузите данные одним из способов:
   - **Загрузить с сервера** — тянет данные напрямую с `urfu.ru/api/entrant/` (может упереться в CORS);
   - **Снипет для консоли** — откройте `urfu.ru/ru/alpha/full`, вставьте снипет со страницы в консоль браузера (F12), скачается `urfu_packed.json` — перетащите его на страницу;
   - **Загрузить файл JSON** — если файл уже есть.
2. Дальше всё считается локально в браузере: фильтры по институту, финансированию, статусу, приоритету и баллу, экспорт в CSV.

Данные никуда не отправляются — приложение полностью статическое.

## Публикация

Сайт публикуется на GitHub Pages автоматически при пуше в `main` (workflow `.github/workflows/deploy.yml`). Сборка не требуется — деплоится статика как есть.
